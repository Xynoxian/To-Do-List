# TaskFlow To-Do List

CS114 Programming II project. Python + CustomTkinter + SQLite.

## Run

```
pip install -r requirements.txt
python main.py
```

## Tests

```
python test_programs/test_program1.py
python test_programs/test_program2.py
```

## Files

- `main.py` - starts the app
- `database.py` / `auth.py` - db + login
- `task_manager.py` - tasks CRUD
- `scheduler.py` - priority / workload stuff
- `analytics.py` - stats and achievements
- `gui/` - all the screens
